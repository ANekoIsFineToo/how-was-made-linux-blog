## ¿Qué tecnología se ha empleado?

---

### Tanto el blog como la presentación estan basados en frameworks de `React` (JavaScript).

<Appear>
    <p style={{ fontSize: '1.25em' }}>Para el blog se ha utilizado un framework llamado GatsbyJS el cual a partir de markdown y React permite generar cualquier tipo de página estática.</p>
    <p style={{ fontSize: '1.25em' }}>En la presentación se ha utilizado otro framework que también funciona mediante markdown y React pero orientado exclusivamente a presentaciones llamado mdx-deck.</p>
</Appear>

---

### Los servicios utilizados para compartir y publicar los cambios realizados por cada miebro son `GitHub` y `Netlify`.

<Appear>
    <p style={{ fontSize: '1.25em' }}>GitHub es un VCS o servicio de repositorios en el cual mediante un cliente de git se pueden enviar cambios persistentes.</p>
    <p style={{ fontSize: '1.25em' }}>Netlify ofrece hosting de páginas estáticas mediante una CDN de alta velocidad, que ademas permite conectarse con GitHub para que cada cambio enviado se publique automaticamente.</p>
</Appear>
